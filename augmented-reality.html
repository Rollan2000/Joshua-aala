Augmented Reality: AR Animation

Overview

To animate the Posters into a 2D looping animation and see how it moves in real life in augmented reality

Roles

Adobe Photoshop, Meta Spark Studio, Meta Spark Player

Tools

AR designer, AR Developer

Research Proposal: This project aims to animate the Posters into a 2D looping animation and see how it moves in real life in augmented reality.

The animation will be rendered using 2D animation software and will be integrated into a mobile app that allows users to view the animation in real life through augmented reality.

Project Goal: To animate the Movie poster, Game cover and Book cover into a 2D looping animation and see how it moves in real life in augmented reality. The animation will be rendered using 2D animation software.

The animation will be integrated into a mobile app that allows users to view the animation in real life through augmented reality. The app will also provide users with an immersive experience.

Tasks & plan:

    1. Redesign the Poster of the Movie poster, Game cover and Book cover in Adobe Photoshop using the patch tool, Quick selection tool, Layer placements, colour adjustments, etc.
    2. Transfer all Exported PNG objects to Meta Spark Studio.
    3. Use Target Tracker to Scan the Poster (Game, book, movie cover) to animate.
    4. Render in Facebook, Instagram , Meta Spark player Mobile App.
    5. Scan the target Poster, Game Cover or Book Cover to see the AR animation experience.

1. AR Experience Movie Poster: Transformers

2. AR Experience Game Cover: Sonic Adventure 2

3. AR Experience Book Cover: Star Wars Republic Commando

4. AR Experience Game Cover: Transformers - Fall of Cybertron
