Motion Graphics Design: Nintendo Switch

Tool(s) used: Adobe Illustrator, After Effects

Video 1 Video 2 Video 3 Bonus Video

Research Proposal: The Nintendo Switch is a hybrid console created by the Nintendo company. Basically, it is a tablet that can be used as a gaming console or as a tablet that can be used as a handheld gamepad. It was released worldwide on March 3, 2017.

This project was inspired by this opportunity. Back in my previous semester, I learned more about designing logo brands in After Effects, and I want these skills to help me become a better motion designer.

Project Goal: The main objective of this project is to create three 15-second animations using Adobe After Effects promoting the Nintendo Switch brand, export MP4 renders for each animation, and link them to YouTube.
