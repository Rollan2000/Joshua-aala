Motion Graphics Design: Nintendo Switch

Overview

Creating a motion design series of three 15-second animations of a “Nintendo Switch product Motion Graphics Ad Campaign”.

Roles

Motion Graphic Designer

Tools

Adobe Illustrator, After Effect,Youtube, MP4

Research Proposal: The Nintendo Switch is a hybrid console created by the Nintendo company. Basically, it is a tablet that can be used as a gaming console or as a tablet that can be used as a handheld gamepad. It was released worldwide on March 3, 2017. This project was inspired by this opportunity. Back in my previous semester, I learned more about designing logo brands in After Effects, and I want these skills to help me become a better motion designer.

Project Goal: The main objective of this project is to create three 15-second animations using Adobe After Effects promoting the Nintendo Switch brand, export MP4 renders for each animation, and link them to YouTube.

I planned to promote the Nintendo Switch features including a Switch console, promote games, and customization of Joy-Con colours.

Tasks & plan:

    1. Research for more Nintendo switch product motion graphic inspiration videos.
    2. Create a mood board design,or brand aesthetics, and style.
    3. Create a storyboard using rough draft paper.
    4. Create Nintendo switch designs using Illustrator for three videos.
    5. Create 3 style frames for each three Switch products.

Brand aesthetics and style

As I reviewed my Brand aesthetics and styles, I had to draw storyboards using pen and paper for every three videos to serve as visual guides through the production process before moving on to After Effects. I sketched three storyboards for Nintendo Switch: Feature System, Color customization, and Game promotion.
